🚀 Project Reflection

 1. Chosen Framework:
For this project I decided to use Bootstrap.

2. (Why did you choose this framework for this specific business?) 

        I chose Bootstrap because it already has built-in styles and components, so it made it easier and faster for me to add things like buttons without having to design everything from scratch. The buttons are affected because Bootstrap automatically applies its own styling (like colors, padding, and hover effects) through the classes I added, so they look more clean and consistent across my site.

4. The Responsive Challenge:
   
        The hardest part was getting the navbar to fit on mobile instead of staying in one long line like it does on desktop. I fixed it by using Bootstrap’s responsive navbar, which automatically collapses into a menu button on smaller screens so everything still fits and looks clean.

6. Semantic Choices:

        I used semantic tags like <section> instead of just <div> because they actually describe what the content is. It helps organize the code better and makes it easier to read, and it can also help with things like accessibility and SEO.
